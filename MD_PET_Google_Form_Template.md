# MD PET Website - Client Data Collection Form
## Google Form Template Guide

Use this guide to create a Google Form. Copy each section and question into your Google Form.

---

## 📝 FORM TITLE & DESCRIPTION

**Title:** MD PET Website - Data Collection Form

**Description:** 
```
Dear Client,

We are building your professional website and need accurate information to make it complete and production-ready. 

Please fill out this form with all the required details. This will help us create a website that truly represents your business.

Estimated time: 30-45 minutes
Tip: Keep your brochures and company documents handy while filling this form.

Thank you!
```

## 📌 SECTION 1: COMPANY INFORMATION

**Section Title:** Company Information
**Section Description:** Basic details about your company

### Questions:

1. **Company Name** (Required)
   - Type: Short answer
   - Example: "MD PET Industries Pvt. Ltd."

2. **Company Tagline/Slogan** (Required)
   - Type: Short answer
   - Placeholder: "e.g., Complete PET Bottle Manufacturing Solutions"

3. **Year of Establishment** (Required)
   - Type: Short answer
   - Validation: Number

4. **Brief Company Description** (Required)
   - Type: Paragraph
   - Description: "Write 2-3 sentences about your company (will be used on About page)"

5. **Total Machines Installed Till Date** (Required)
   - Type: Short answer
   - Example: "2500+"

6. **Number of Countries Exported To** (Required)
   - Type: Short answer
   - Example: "45+"

7. **Total Happy Clients/Customers** (Required)
   - Type: Short answer
   - Example: "500+"

8. **Team Size (Number of Employees)** (Required)
   - Type: Short answer
   - Example: "100+"

9. **Factory/Manufacturing Area Size** (Optional)
   - Type: Short answer
   - Example: "50,000 sq ft"

10. **ISO/Quality Certifications** (Optional)
    - Type: Paragraph
    - Description: "List any certifications with year obtained (e.g., ISO 9001:2015)"

## 📌 SECTION 5: PRODUCTS - MACHINE 1

**Section Title:** Product Details - Machine 1
**Section Description:** Fill details for your first machine. Repeat for each product.

### Questions:

32. **Machine/Product Name** (Required)
    - Type: Short answer
    - Example: "3 Cavity Fully Automatic PET Blow Molding Machine"

33. **Product Category** (Required)
    - Type: Multiple choice
    - Options:
      - Semi Automatic
      - Fully Automatic
      - Eco/Budget Series
      - Electra/Servo Series
      - Moulds
      - Auxiliary Equipment
      - Other

34. **Production Capacity (Bottles Per Hour)** (Required)
    - Type: Short answer
    - Example: "2800-3000 BPH"

35. **Maximum Bottle Volume** (Required)
    - Type: Short answer
    - Example: "2 Liters" or "2000ml"

36. **Number of Cavities** (Required)
    - Type: Short answer
    - Example: "3" or "N/A for moulds"

37. **Short Description (1-2 sentences)** (Required)
    - Type: Paragraph
    - Description: "Brief description for product card"

38. **Detailed Description (3-4 sentences)** (Required)
    - Type: Paragraph
    - Description: "Detailed description for product page"

39. **Key Features (List 6-8 features)** (Required)
    - Type: Paragraph
    - Description: "Enter each feature on a new line"
    - Example:
      ```
      Fully automatic operation
      Servo-driven stretching
      PLC with HMI touch screen
      Energy efficient design
      Quick mould changeover
      Auto fault detection
      ```

40. **Technical Specifications** (Required)
    - Type: Paragraph
    - Description: "Enter in format: Specification Name: Value (one per line)"
    - Example:
      ```
      Output Capacity: 2800-3000 BPH
      Max Bottle Volume: 2000ml
      Number of Cavities: 3
      Preform Neck Size: 28mm / 38mm
      Max Bottle Diameter: 110mm
      Max Bottle Height: 335mm
      Heating Power: 45 KW
      Compressed Air: 25-35 Bar
      Machine Weight: 3500 kg
      Dimensions: 4500 x 1800 x 2200 mm
      ```

41. **Upload Product Images (2-3 images)** (Required)
    - Type: File upload
    - Description: "High resolution, different angles"
    - Allow multiple files

42. **YouTube Video Link for this Product** (Optional)
    - Type: Short answer
    - Description: "YouTube video URL showing this machine"

43. **Price Display Preference** (Required)
    - Type: Multiple choice
    - Options:
      - Show "Price on Request"
      - Show actual price/range: ____
      - Don't show price

44. **Is this a Featured/Popular Product?** (Required)
    - Type: Multiple choice
    - Options:
      - Yes, feature this product
      - No, regular listing

## ✅ FORM SETTINGS TO CONFIGURE

In Google Forms Settings:
1. ✅ Collect email addresses
2. ✅ Allow file uploads (increase to 1GB limit)
3. ✅ Send respondent a copy of their response
4. ✅ Show progress bar
5. ✅ Shuffle question order: OFF
6. ✅ Confirmation message: "Thank you! We've received your information and will contact you within 48 hours."

## 🔗 QUICK GOOGLE FORM CREATION LINK

Go to: https://forms.google.com/create

Steps:
1. Click "Blank" to create new form
2. Copy title and description from above
3. Add sections using the "Add section" button (Tt icon)
4. Add questions from each section
5. Set required fields as marked
6. Enable file uploads in Settings > General > "Allow file uploads"
7. Preview and test before sharing
